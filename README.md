## Installation

### From source

```shell
git clone git@github.com:marianocarrazana/dockery.git
cd dockery
pip install -e .
# update only:
git pull
```

### From pip

```shell
pip install dockery
```

## Usage

Run on your console:

```shell
dockery
```

## Extra commands

```shell
dockery df
dockery ps
dockery volumes
dockery images
dockery stats
```
