Graphical interface for Docker in your console

## Installation

### From source

```bash
git clone git@github.com:marianocarrazana/dockery.git
cd dockery
pip install -e .
# update only:
git pull
```

### From pip

```bash
pip install dockery
```

## Usage

Run on your console:

```bash
dockery
```

## Extra commands

```bash
dockery df
dockery ps
dockery volumes
dockery images
dockery stats
```
